# 行为准则

本仓库服务于学术研究、代码复现和教学交流。参与讨论、提交 Issue 或 Pull Request 时，请遵守以下基本规则。

## 1. 期望行为

- 尊重事实和证据，围绕代码、数据和文档本身讨论
- 明确区分“已验证结论”“待验证猜想”“个人建议”
- 对复现失败、结果差异和潜在错误保持具体、可检查的表达
- 尊重不同背景的参与者，避免人身化表达

## 2. 不可接受的行为

- 人身攻击、羞辱、歧视或恶意嘲讽
- 在无证据情况下指控学术不端或恶意行为
- 公开传播可能违反许可的数据、文档或第三方材料
- 刻意刷屏、重复争论或偏离仓库主题

## 3. 学术讨论补充原则

本仓库鼓励严谨质疑，但不鼓励空泛争吵。对以下内容，建议提供最小可核查证据：

- 数值不一致
- 统计方法适用性问题
- 路径设计或变量定义争议
- 结果图和论文表述不一致

## 4. 执行方式

维护者可对明显不符合本准则的内容进行：

- 请求修改
- 锁定讨论
- 关闭 Issue 或 PR
- 限制进一步互动

## 5. 安全与合规问题

若问题涉及数据许可、隐私风险或潜在安全缺陷，请优先按 [SECURITY.md](SECURITY.md) 处理，不建议直接在公开 Issue 中披露细节。
