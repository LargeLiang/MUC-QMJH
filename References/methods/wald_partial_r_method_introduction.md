# Wald 部分相关 r 入门

## 一句话理解

Wald 部分相关 r 是一种从回归系数和标准误出发构造的局部效应量，用来衡量某个回归系数在控制其他变量后有多强。

## 本项目在哪里使用

- C18 在长度和格式的嵌套逻辑回归中都报告了 Wald 部分相关 r。

## 本项目中的计算方式

代码采用的近似形式是：

r = z_wald / sqrt(N)

其中：

- z_wald = beta / SE
- N 是样本量

这意味着：

- 系数越大、标准误越小，r 往往越大。
- 样本越大，同样的 z 值换算成 r 时会更保守一些。

## 它在问什么

当模型里已经控制了很多变量后，我们仍然想知道：

“核心变量在剩余解释空间中，局部影响大不大？”

Wald 部分相关 r 就是在做这件事。

## 和普通相关系数有什么区别

- 普通相关系数通常来自两个变量的直接共同变化。
- Wald 部分相关 r 来自回归模型，是“控制其他变量之后”的局部强度。

所以它更贴近“净效应”的语境。

## 如何读结果

如果某个调整后模型中的 Wald r 仍保持正值且不小，通常说明：

- 这个变量在控制其他混淆后，仍保留了稳定的局部影响。

如果粗模型 r 较大，而控制后明显下降，则说明部分效应可能被混淆因素解释掉了。

## 为什么项目里要报告它

因为 OR 更偏向相对赔率解释，AME 更偏向概率变化解释，而 Wald r 可以补充一个“局部效应强度”的视角。三者放在一起，比只报一种指标更完整。

## 零基础常见误区

- 它不是 Pearson 相关。
- 它不是标准化回归系数本身。
- 它更适合做相对比较，而不是机械套用固定阈值。

## 在本研究中的作用

本项目用它来监控长度和格式变量在嵌套控制下的“净强度”是否保留。它相当于为回归结果再补一层效应量表达。
