## 一、立项依据

### 1.1 现状与背景分析
- RLHF 主流范式下的无意识偏好（长度偏好、格式偏好）
- 形式与质量的伪相关性 → 过优化 / reward hacking
- 现有研究局限：重“如何对齐”，轻“对齐什么”
- 研究定位：前移焦点，系统揭示、量化、验证偏好及其影响机制

### 1.2 学术与应用价值
1. 指导高质量训练数据集的构建与清洗
2. 为人类偏好研究提供“多变量机制建模”方法论参考（SEM路径分析）

### 1.3 已有研究实践基础
- **数据基础**：LMArena 的 arena-human-preference-140k 数据集
  - 13.6万条标注记录，53种模型
  - 关键字段：id, evaluation_session_id, winner, conversation_a/b, category_tag 等
- **指导支撑**：程豪老师（统计计算、结构方程模型方向）

### 1.4 团队优势
- 统计学专业本科生构成，知识与能力互补
- 数据处理、模型构建、结果分析全流程协作能力

## 二、研究内容、目标、拟解决的关键问题

### 2.1 研究内容（四大模块）

#### 模块1：数据初步处理与描述性统计
- **长度与格式特征量化编码**
  - 长度：Token数、长度比率、长度差异
  - 格式：标题数、列表数（有序/无序）、粗体数
- **数据集拆分及描述性统计**
  - 按 category_tag 拆分为数学、创意写作、指令遵循等子集
  - 可视化：双轴折线图、小提琴图、子图矩阵热力图
- **产出具体假设**（如长度比率区间偏好、列表类型偏好）

#### 模块2：人类偏好的存在性检验与强度量化
- **存在性检验**
  - 长度偏好：Mann-Whitney U 检验 / ANOVA / Kruskal-Wallis
  - 格式偏好：卡方检验 / Fisher 精确检验 / 残差分析
- **效应量计算**
  - Cramér‘s V, OR, Cohen’s d, η²
  - 结合强度判定标准与实践意义解释

#### 模块3：伪相关性建模与控制混淆变量
- **核心混淆变量**
  - 模型能力等级差（高/中/低）
  - 问题类型（category_tag 哑变量）
- **分层逻辑回归建模**
  - 仅控制变量 → 加入长度/格式特征 → 净效应判断
- **混淆控制效果验证**
  - 按问题类型分层、按模型能力分层交叉验证
- **净效应量化**

#### 模块4：偏好影响机制与路径探索
- **变量定义**
  - X1（长度特征）、X2（格式特征）
  - Y（偏好选择，二分类）
  - M1（感知体验层：易读性、情感导向）
  - M2（语义质量层：语义连贯性、信息密度、人工标注特征）
  - C（模型能力等级差、问题类型哑变量）
- **分层中介效应四步检验**
- **SEM路径假设与扩展分析**
  - 外生变量、中介变量、内生变量、控制变量、调节变量（任务类型）

### 2.2 研究目标
1. 证实或证伪长度与格式偏好的系统性存在
2. 精确量化不同模型与任务类型下的效应大小
3. 揭示并控制关键混淆变量
4. 通过 SEM 确定偏好形成机制，分离形式与内容质量影响

### 2.3 创新点
- 视角创新：前移至“人类偏好数据是否存在系统性偏差”
- 方法创新：统计因果推断方法系统性应用于 LLM 偏好研究
- 解释创新：SEM 多路径分析 + “感知体验层 - 语义质量层”分层中介
- 问题导向创新：直面 RLHF 中的 reward hacking 和 over-optimization

### 2.4 拟解决的关键问题
1. 混淆变量的有效控制（利用 category_tag、模型胜率，分层回归）
2. 偏好强度的可操作化度量（效应量 + 偏好偏差指数）
3. 复杂影响路径的建模与解释（从简到繁，优先稳健性与可解释性）
