# Bootstrap 置信区间入门

## 一句话理解

Bootstrap 是一种重抽样方法。它通过“从原始样本中反复有放回抽样”，来近似估计某个统计量的波动范围，从而构造置信区间。

## 本项目在哪里使用

- C16 和 C17 用它给中位差或相关效应提供 95% 区间。
- C19 和 C20 用它给 IPW ATE 提供 95% 区间。
- C22 用它给直接效应、间接效应和总效应提供 95% 区间。

## 为什么需要 Bootstrap

很多统计量都不是简单均值，特别是：

- 中位数差
- 比值类效应
- 间接效应这种乘积项

这些量的抽样分布不一定接近正态。如果还强行套用传统正态近似，区间可能不稳。Bootstrap 的优点就是：少依赖分布假设。

## 它是怎么做的

以原始样本为基础：

1. 有放回抽一份和原样本一样大的新样本。
2. 在这份新样本上重新计算目标统计量。
3. 重复很多次，得到一串统计量。
4. 取这串统计量的 2.5% 和 97.5% 分位数，形成 95% 百分位区间。

## 本项目中的典型理解

如果某个间接效应的 bootstrap 95% CI 为 [0.01, 0.08]，这意味着：

- 在重复重抽样下，这个效应的合理范围大致落在 0.01 到 0.08。
- 因为整个区间都大于 0，所以通常认为它更支持“正向效应存在”。

如果区间跨过 0，就说明方向还不够稳定。

## 为什么它特别适合本项目

本项目里有不少复杂统计量：

- 配对差异下的区间估计
- IPW 加权后的 ATE
- SEM 中的间接效应

这些都比简单均值复杂，Bootstrap 可以提供更稳健的补充证据。

## 零基础常见误区

- Bootstrap 不是重复收集新数据，它只是在现有样本上重抽样。
- Bootstrap 结果仍然依赖原始样本质量，原始样本有偏，Bootstrap 不能自动修正。
- “区间不跨 0”是很常见的判断方式，但仍要结合模型设定和样本代表性。

## 在本研究中的作用

Bootstrap 是项目里重要的“稳健区间层”。它特别适合给复杂效应一个比单点估计更可靠的解释范围，让报告不只停留在“某个数是 0.043”，而是告诉读者这个数有多稳定。
